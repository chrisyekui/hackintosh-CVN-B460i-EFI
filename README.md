# hackintosh--CVN-B460I-EFI
EFI的OC版本为0.83，适用于Ventura beta3网卡AX200，配置一切完美，HDMI接口已定制
安装前
开启：1.设置为IGX
     2.SATA设置➡️AHCI
     3.USB➡️XHCI交接
关闭：
     1.ME控制
     2.BIOS写保护
     3.C状态
     4.cfg lock
     5.CSM support
其他版本需要更换Wi-Fi和蓝牙驱动,以后应该会更新EFI,有问题请发到syuyx@foxmail.com或者GitHub的问题里，有空我都会看的。
